Die Tasten-Bedienung ist für die Spiel-Entwicklung vorteilhaft gewesen. So kann man besser Fehler finden.
Bei Eingabe von maloi deckt der Computer seine Karten auf
Bei drücken der Tasten Strg+N kann das aktuelle Spiel mit den gleichen Karten noch einmal gespielt werden
Bei drücken der Tasten Strg+S wird der Lorum-Spielbogen angezeigt
Bei drücken der Tasten Strg+E kommt eine MsgBox ob das Spiel abgebrochen werden soll
